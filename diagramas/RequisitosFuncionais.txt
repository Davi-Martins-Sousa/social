RF01 – O sistema deve permitir o cadastro de novos usuários.
RF02 – O sistema deve permitir que o usuário faça login utilizando e-mail e senha.
RF03 – O sistema deve permitir que o usuário edite seu nome e e-mail.
RF04 – O sistema deve permitir que o usuário siga outro usuário.
RF05 – O sistema deve permitir que o usuário pare de seguir outro usuário.
RF06 – O sistema deve permitir que o usuário remova um seguidor.
RF07 – O sistema deve permitir que o usuário remova alguém que ele está seguindo.
RF08 – O sistema deve permitir que o usuário crie postagens.
RF09 – O sistema deve permitir que o usuário edite o conteúdo de suas próprias postagens.
RF10 – O sistema deve permitir que o usuário exclua suas próprias postagens.
RF11 – O sistema deve permitir que o usuário comente em uma postagem.
RF12 – O sistema deve permitir a exclusão de um comentário pelo seu autor.
RF13 – O sistema deve permitir visualizar os comentários de uma postagem.
RF14 – O sistema deve exibir um feed com postagens dos usuários seguidos.
RF15 – O sistema deve ordenar o feed por data de criação das postagens (mais recentes primeiro).
RF16 – O sistema deve permitir a pesquisa de usuários por nome.
RF17 – O sistema deve exibir a lista de usuários que um usuário está seguindo.
RF18 – O sistema deve exibir a lista de seguidores de um usuário.
RF19 – O sistema deve notificar o usuário quando for seguido por outro.
RF20 – O sistema deve permitir ao usuário visualizar uma postagem em detalhe, incluindo comentários.
RF21 – O sistema deve associar cada postagem ao seu autor.
RF22 – O sistema deve associar cada comentário ao seu autor e à postagem correspondente.
RF23 – O sistema deve validar os dados inseridos no cadastro (ex: e-mail válido).
RF24 – O sistema deve manter a sessão ativa após login até que o usuário saia manualmente.
RF25 – O sistema deve permitir logout do usuário.
RF26 – O sistema deve impedir que um usuário edite ou delete postagens de outro usuário.
RF27 – O sistema deve impedir que um usuário edite ou delete comentários de outro usuário.
RF28 – O sistema deve permitir a visualização do perfil de qualquer usuário.
RF29 – O sistema deve exibir a quantidade de seguidores e seguindo no perfil.
RF30 – O sistema deve permitir ao usuário denunciar uma postagem ou comentário inadequado.
